# Referral Dashboard (React)

Simple ReactJS referral dashboard assignment.

Run locally:

```bash
cd benchmark_assignment
npm install
npm run dev
```

Open http://localhost:5173

Features:
- Header with nav, search, avatar
- Stats cards (mocked)
- Referral link/code with copy
- Referrals table (API-driven) with search and pagination (10 rows/page)
